# Scramble 1

`R' U' F D R F2 D L F D2 F2 L' U R' L2 D' R2 F2 R2 D L2 U2 R' U' F`

U B' L2 F                                   // eo   (4/4)
(L2 U) R D' F2 B2 D2 R2 D' L2 U D2 R        // dr   (13/17) lol
(D' B2 D' L2 D')                            // htr  (5/22)
(B2 L2 R2 F2 R2 U2 F2 U2 F2 D2)             // fin  (10/32)
