# Scramble 2

`R' U' F L' F2 D B2 U2 B2 F2 R2 D R2 F R2 D R2 D' B L' R U R' U' F`

## Solution

```
(B)                         // eo   (1/1)
R D' (D2 R D2 L' F2 L' R D) // dr   (10/11)
D2 L (D2 R' D2 F2 R B2 R')  // htr  (9-2/18)
(U2 L2 D2 F2 L2 *)          // 2e2e (5/23)

* L2 D2 F2 L2 F2 D2 L2 F2
```
