# Scramble 2

`R' U' F L F2 L2 B U2 B' D2 F2 L2 F D2 L F' D U2 F2 D2 L D2 U R' U' F`

## Solution

```
L' R F' R'                          // eo           (4/4)
F2 B2 U' D F                        // dr           (5/9)
D' @ F2 D' R2 D2 # L2 B2 D @ F2 D'  // htr          (10/19)
L2 B2                               // 2e2e-slice   (2/21)

@ E                                 //              (1/22)
# E2                                //              (0/22)
```

`L' R F' R' F2 B2 U' D F U D2 L2 D' F2 U2 F2 L2 U F2 D' L2 B2 (22)`
