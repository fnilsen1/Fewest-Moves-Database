`R' U' F U B2 D' F' D R2 U B' R2 L2 F R' B2 L2 F2 U2 L2 U2 B2 R' U' F `

## Solution

B L D2 R2 L2 B R2 B         // dr   (8/12)
(R') B2 D2 L D2 L'          // htr  (6-1/17)
L2 * B2 D2 L2 B2            // 2e2e (5-1/21)

* D2 B2 L2 D2 B2 D2 L2 B2   //      (8-2/27)
